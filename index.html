⚠️ Instrument Calibration Correction — Tiger Analyzer (active)

Ralf Aben (Radboud University) identified a substantial calibration deviation in the Tiger microportable CO₂/CH₄ analyzer. The following linear corrections are applied to all dry-mole-fraction readings before any flux calculation:

CH₄_corrected = 0.97377900 × [CH₄]d_ppm + 0.01870000
CO₂_corrected = 0.86616349 × [CO₂]d_ppm + 5.26790000

These equations were derived from the Example_of_correction.xlsx file (R² = 1.000 for both). The correction is linear, using the dry-mole-fraction columns ([CH₄]d_ppm and [CO2]d_ppm) as input. The CO₂ deviation is substantial (~12.5% bias in slope); the CH₄ deviation is smaller (~2.6%). H₂O is not corrected by this equation. Always use corrected values for flux calculations.
